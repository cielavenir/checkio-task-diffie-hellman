Implement Diffie Hellman key exchange.

Input: Three integers: prime, root-modulo, exchange number. Note that prime and root-modulo are predefined but exchange number is random.

Output: Tuple or List of two integers: another exchange number and the secret key. (Note: Secret key is returned because this is coding problem)

Example: 

checkio(23,5,19) # => (8,2) # example in https://en.wikipedia.org/wiki/Diffie%E2%80%93Hellman_key_exchange (checkio server is assumed to be Alice)

How it is used: Basic cryptography.
